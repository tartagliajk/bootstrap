Code snippets

Code snippets är mallar som gör det enklare att ange upprepade kodmönster, såsom loopar eller villkorssatser. Det finns både inbyggda och nerladdningsbara.

WASM

Web assembly, gör det möjligt att skriva på webben i andra språk förutom html, css och javacript dvs bygga mjukvara med c+, rust och python och sedan skicka de till en användare utan att de behöver ladda ner något.

Chromium

En webbläsare skapad från en öppen källkod och har lagt grund till dagens webbläsare (google chrome, microsoft edge etc. Den har ungefär samma funktioner som google chrome, dock så uppdateras den inte längre.

svg

Scalable Vector Graphics, vektorformat för tvådimensionella bilder som stödjer animationer. Skillnaden mellan svg och exempelvis png är att den är ritade av vektorer och inte bitmap vilket har sin fördel att den blir tydligare i detalj.

png24

Supportar 24-but färger, vilket leder till mer färg tillgänligheter än png8. PNG 24 används bäst för webbfoton och mer detaljerade bilder.
