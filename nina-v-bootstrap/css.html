Local vs CDN bootstrap

Har att göra med webbservrar som använder sig av bootstrap. CDN är ett sätt att förkorta vägen för information mellan användaren och servern. Dock används vissa lokalt, detta kan däremot sakta ner informationsvägen. CDN kan sänka kostanden för bredband (då mindre data används), i slut ändan brukar folk använda båda två samtidigt.

JQuery

Är ett javascript biblotek som i grunden är menat att förenkla HTML-, DOM-, CSS-modifikation etc. Detta göt så att program som egentligen kräver många rader javascript kod behöver endast en med hjälp av JQuery.

Flexbox

En module som enklare behandlar sätt att fördela utrymmet mellan föremål i en container, även när dess storlek är okänd och/eller dynamisk. Finns olika kommando, några exempel är flex-grow, flex-wrap, align-content etc.

Grid

Två dimenstionellt rutnätbaserat layoutsystem. Liknande princip som flexbox, därmed funkar dessa två bra tillsammans. Vanliga kommandon är grid-column-start, grid-area, justify-self etc.
