HTML (generellt)

(HyperText Markup Language) Programmeringsspråk som används för att strukturera och visa en hemsida och dess innehåll. Dvs man använder HTML element för att strukturera

HTML Element

Elements som ändrar innehållet på olika sätt. Det vill säga man kan ändra så att vanlig text är en rubrik, lista eller att texten länkar till något. Elementen har en start tag --> kontent --> slut tag. Struktur/formell: "tag-namn" Kontent "/tag-namn"

ID

Specifik/unik id för ett HTML element. Används ofta för att deklarera ett element till en viss style i css. Därmed kan man manipulera element med specifika id.

Class

Används för att pointera ett visst class namn i style filen, man kan använda fler än 1. Används inte bara av html, utan även CSS och javascript. Class namn måste skrivas exakt och därmed känsliga för versaler och gemener.

Meta (ordet)

Metadata, tagen skrivs som "meta" och definierar metadata (information om data) om ett HTML dokument. Tagen finns alltid i head elementet och används oftast för ex sid info, nyckelord etcetera. Metadatan är inte synlig på hemsidan.

Responsive

Innebär att man skapar websidor som ser bra ut på alla enheter, görs genom att den automatiskt ändrar utseende efter skärmstorlek och synpunkter. För att lägga det på en hemsida så läggs: ” ’meta name="viewport" content="width=device-width, initial-scale=1.0">”.
