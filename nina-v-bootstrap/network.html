HTTP

Hypertext transfer protocol

  1. Osäker, ingen data encryptning
  2. Appliceringsskikt
  3. Möjliggör anslutning på begäran
  4. Informationen du får är sårbar

HTTPS

Hypertext Transfer Protocol Secure

  1. Säker
  2. Transportlagersäkerhet (TLS)
  3. Uppgifterna skickas inte som klartext

DNS

Domain name system

  • Låt oss användare ansluta till webbplatser med domännamn istället för IP-adresser
  • "Internets telefonbok"
  • Konverterar värdnamn (www.ex.com) till datorvänlig IP-adress (ex 192.168.1.1)

CDN

Innehållsleveransnätverk

Ett distribuerat nätverk av servrar som effektivt kan skicka innehåll på websidor till användare.

Compressed data

När data "omkodas" för att göra färre informationsbärande enheter används. Användbart då man vill spara lagringsutrymme.
